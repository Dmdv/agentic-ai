# Technical Specification
## Goal
Create a new Python project in the `test_python_skills` directory following the `python-guidelines` skill. This includes initializing the `pyproject.toml`, configuring `ruff`, `mypy`, and setting up the test structure.

## Architecture
1. **Directory Structure**: Create the necessary directories and files for the project.
2. **pyproject.toml**: Initialize the project with a `pyproject.toml` file that includes build and tool configurations.
3. **Ruff Configuration**: Set up `ruff` for linting and formatting.
4. **Mypy Configuration**: Set up `mypy` for static type checking.
5. **Test Structure**: Create a test directory and initial test files.

## Steps
1. Create the `test_python_skills` directory.
2. Initialize the `pyproject.toml` file with the necessary configurations.
3. Create the `ruff` configuration file.
4. Create the `mypy` configuration file.
5. Set up the test directory and initial test files.
